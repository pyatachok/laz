Админ-интерфейс позволяет:
1) Управлять юзерами:
1.1) Существует несколько категорию юзеров. 3 из них 
имеют доступ к редактированию в админ интерфейсе, 
это manager(управление заказами, просмотр залов, сеансов), 
supervisor(управление заказами, залами, схемами, сеансами), 
admin(имеет неограниченые права в админинтерфейсе.)
1.2) Для добавления юзеров существует ссылка, которая ведет на страницу 
http://liveanimation.ru/admin/users/add
Юзер добавляется неактивированным, нужно активировать, 
активировать можно через переход по ссылке в письме на указанный e-mail юзера,
или менеджером в админке. В админке также присутствует 
кнопка Resend leter (отправляет ссылку для активации ещё раз).
Также присутствуют кнопки редактирования и удаления юзера.
2) Просмотр старых заказов http://liveanimation.ru/admin/bron
3) Управление залами http://liveanimation.ru/admin/zal
Делится на несколько этапов: создание зала, редактирование схемы зала.
1) Создание зала - перейти по ссылке "Добавить новый зал"
http://liveanimation.ru/admin/zal/add
!!! Псевдоним нужно вводить латиницей
!!! Количество рядов для частей зала необходимо задать сразу, 
при создании зала, в дальнейшем не редактируется.

Высота и ширина схемы в пикселях нужны для настройки отображения 
схемы в попап окне при выборе мест в зале.

После создания зала необходимо отредактировать схему зала, 
для этого перейдите по ссылке - "Редактирование схемы зала" для выбранного зала.
Редактирование категорий мест в зале - позволяет группировать места для дальнейшего назначения цен.
Редактирование геометрии зала - позволяет фигурно изменять геометрию зала, 
например добавить проходы, или создавать сложные контуры для частей зала.

4) Управление сеансами
http://liveanimation.ru/admin/seans
Для добавления нового сеанса - перейдите по ссылке "Добавить новый сеанс".
Позволяет задать имя, псевдоним(для выбора сеансов на одно представление в одном зале,
для страницы покупки билетов), выбрать зал в котором проводится сеанс, 
задать строгий порядок мест или свободная рассадка, 
задать возможность покупки подарков и стоимость подарков для сеанса.
Далее ВАЖНО нужно в редакторе схемы сеанса задать стоимость мест согласно категориям.
Также в редакторе схемы сеанса можно закрепить места за кассами.
Изменить категории мест для одного сеанса не изменяя категории в зале(для всех сеансов).
Чекбокс - Строгий порядок, позволяет вывести на схеме зала, 
предупреждение для клиентов что выбор места на схеме не влияет на рассадку в зале.

5) Управление заказами http://liveanimation.ru/admin/order
Позволяет распечатать билет - ссылка ряд-место билета
Кнопка "отменить" Позволяет отменить заказ - Освобождает места.
Кнопка "Купить" Переводит заказ в состояние - Куплено.
Помните! Действия производимые вами могут иметь последствия!
Кнопка "Удалить" НЕ освобождает места на схеме. 
Будьте осторожны с внесением изменений
